Point of Sales Project | C#

Point of sale terminal used to order and purchase products. Utilizes user input validation to ensure valid results. Structured with classes for organization. Takes advantage of OOP principles such as inheritance and polymorphism through an abstract class. Includes easy customization such as a product list generated from a csv file that may be edited or replaced.

Ice Cream Votes Lab | Java

App that allows each user of a family set to pick a favorite from a list of ice cream flavors. Utilizes user input validation to ensure valid results. Structured with classes for organization. Displays the most voted for flavor at the end (multiple in case of a tie).
